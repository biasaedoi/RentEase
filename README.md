# RentEase — Aplikasi Peminjaman Barang

**RentEase** adalah aplikasi web berbasis **Laravel** yang dirancang untuk memudahkan proses peminjaman barang antara pengguna dan admin.  
Dengan sistem ini, pengguna dapat melihat daftar barang yang tersedia, mengajukan penyewaan, dan memantau status pinjaman semuanya dalam satu platform yang sederhana dan intuitif.

---

# Fitur Utama

# Pengguna
- Melihat daftar barang yang tersedia untuk disewa.
- Melihat detail setiap barang (gambar, deskripsi, harga per hari, dan status).
- Mengajukan penyewaan dengan memilih tanggal mulai dan selesai.
- Melihat status penyewaan mereka.

# Admin
- Menambahkan, mengedit, dan menghapus data barang.
- Melihat daftar penyewaan aktif dan riwayat transaksi.
- Tidak bisa mengajukan sewa (khusus pelanggan saja).
